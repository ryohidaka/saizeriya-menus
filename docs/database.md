# データベース構成

## データ構造

### カテゴリマスタ (`categories`)

| KEY 名    | 説明               | データ型  | サンプル     |
| --------- | ------------------ | --------- | ------------ |
| `id`      | カテゴリ ID        | `INTEGER` | `1`          |
| `name`    | カテゴリ名         | `TEXT`    | `グランド`   |
| `name_en` | カテゴリ名(英語)   | `TEXT`    | `Grand Menu` |
| `name_zh` | カテゴリ名(中国語) | `TEXT`    | `主`         |

### ジャンルマスタ (`genres`)

| KEY 名 | 説明        | データ型  | サンプル |
| ------ | ----------- | --------- | -------- |
| `id`   | ジャンル ID | `INTEGER` | `12`     |
| `name` | ジャンル名  | `TEXT`    | `サラダ` |
